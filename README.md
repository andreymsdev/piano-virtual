# Piano Virtual com HTML, CSS e JavaScript

 Este projeto é um piano virtual interativo desenvolvido com **HTML**, **CSS** e **JavaScript**. 

## Funcionalidades

- **Interatividade**: Toque as teclas clicando no mouse ou pressionando as teclas do seu teclado.
- **Controle de Volume**: Ajuste o volume do piano conforme sua preferência.
- **Mostrar/Ocultar Nome das Teclas**: Alterna entre a visualização do nome das teclas ou não.
- **Design Responsivo**: Funciona bem em dispositivos móveis e desktops.
- **Toque de Notas com Sons `.wav`**: Sons realistas ao tocar as teclas.

##  Tecnologias Usadas

- **HTMl**: Esqueleto da página
- **CSS3**: Estilização das teclas do piano e responsividade.
- **JavaScript**: Lógica de interação do teclado, controle do volume e eventos de toque
- **Áudio `.wav`**: Sons importados para cada tecla
